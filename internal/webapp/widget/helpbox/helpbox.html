Use tmux to get automatic code block execution

Serve the content locally with mdrip: 

  GOBIN=$TMPDIR go install github.com/monopole/mdrip
  $TMPDIR/mdrip demo {{.AppState.DataSource}}

and run tmux: 

  tmux

Then, at localhost:8080, whatever action copies a code block (↵ or mouse click) also pastes the block to the active tmux pane for immediate execution.
